# Examples

This folder contains simple examples demonstrating how to extend the starter
template created by `init-artifact.sh`. Copy these files into your `app/` or
`components/` directory as a starting point and adjust them to fit your needs.

- **accordion/** – A page demonstrating a Radix Accordion styled with the
  theme variables.
- **card-grid/** – A responsive grid of cards using Tailwind utilities and
  hover effects.

To view an example in your project, copy its folder into `app/` and run the
development server. For example:

```bash
cp -r .claude/skills/next-web-artifacts-builder/examples/accordion app/accordion
pnpm run dev
```
