# Assets (logos and images)

Place your frontend assets here so they are served at `/assets/...`.

**Required files:**
- `logo.png` – used on Login, Signup, Forgot Password, Reset Password, Admin Login, Layout sidebar, and Workspace header.
- `favicon.png` – optional; can be used as the browser tab icon.
- `python.png` – used in the Workspace code editor header (Python badge).

**Where to copy to:**  
`frontend/public/assets/`

After adding these files, the Next.js app will serve them at `/assets/logo.png`, `/assets/favicon.png`, and `/assets/python.png`.
